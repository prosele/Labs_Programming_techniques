# Лабораторные работы по курсу "Методы программирования"    

**Лабораторная работа №1: Алгоритмы сортировок**   
1. Генерация 15 наборов данных об услугах IT-компании (название услуги,  ориентировочная стоимость, срок исполнения, размер преоплаты) для сортировки размерности от 100 до 100000.   
2. Реализация на языке программирования C++ сортировки пузырьком, сортировка простыми вставками и шейкер-сортировки.   
3. Перегрузика операторов сравнения (>, <, >=, <=) для сравнения  объектов. Сравнение по полям – стоимость, размер предоплаты,  название услуги.  
4. Замер времени сортировки  каждым алгоритмом и std::sort. Построение графиков зависимости времени сортировки от размерности массива для каждого из алгоритмов сортировки на одной оси координат.
