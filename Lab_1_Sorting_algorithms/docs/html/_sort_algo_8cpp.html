Detailed Description

Замеры времени выполнения трёх квадратичных сортировок и std::sort.

Читает данные из CSV, сортирует пузырьком, вставками, шейкером и std::sort, замеряет время для каждого размера (100...100000), выводит таблицу и сохраняет отсортированные файлы.
