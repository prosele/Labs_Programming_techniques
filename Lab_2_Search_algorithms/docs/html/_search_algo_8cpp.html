Detailed Description

Сравнение времени поиска по названию услуги различными структурами данных.

Реализованы: линейный поиск, бинарное дерево поиска (BST), красно-чёрное дерево (RBT), хеш-таблица с цепочками и std::multimap. Для каждого размера массива (от 100 до 1 000 000) выполняется 1000 случайных запросов, замеряется время поиска (без учёта построения структур). Для хеш-таблицы подсчитывается реальное число коллизий.
